#Getting Start

As a WooCommerce Add-ons plugin, first you have to confirm that, WooCommerce is installed in your site. Also confirm that you have few Products available in your Store. See How to install WooCommerce. And add products to your Store.

If you are a Woo Product Table Pro newbie, these steps will help you understand the essentials of successful product table creating and showing. You will learn about:

  1. Create a New Table on WordPress and Product Table edit screen

  2. Select your desired columns to show in product table

  3. Select basic query

  4. Set few conditions on table

  5. Setup advanced search box and mini filter

  6. Customize your table's design

  7. Configure your table's settings

  8. Make your table responsive

  9. Publish the table then copy and paste table's shortcode where you need

#Column Overview

  1. Column Title: Give the column a nice name

  2. Disable Description: Don't want to show product short description with product title? By checking this you can hide product short description.

  3. Link Options (Only for Product Title/Thumbnail): You can disable, enable product link or you can enable YITH Quickview on product title and thumbnail. (For YITH Quickview you must have to installed and activated YITH Quickview plugin.)

  4. Wrapper Tag: You can select any tag to wrap this column items. For inner items you need to set tag on that specific columns wrapper tag.

  5. Column Inner Items: Select these items to show up that columns data in this column. These inner items could be order by simply drag and drop.

  6. Wrapper Tag Class: Add custom class to wrapper tag.

  7. Checkbox: This checkbox is responsible for enable/disable this column.

  8. Up/Down Arrow: Using this up and down arrow you can move the column one up/down.

  9. Collapse/Expand: To expand and collapse this section.

#Basics Overview

  1. Category Includes: Select categories to show products in the table from these categories.

  2. Product Includes: Select single products to only these selected products show in the product table.

  3. Product Exclude: Select single products to exclude them from the product table.

  4. Category Exclude: Select product categories to exclude their products from the product table.

  5. Tag Includes: Select product tags to include their products in the product table.

  6. Ajax Action (Enable/Disable): Enable or disable ajax on product table.

  7. Ajax for Pagination (Enable/Disable): Enable or disable ajax pagination on product table.

  8. Mini Cart Position: Change mini cart position.

  9. Table Head: You can hide or show table head using this option.

  10. Pagination On/Off: You can disable table's pagination from here.

  11. Checkbox Auto check on table Load: Enable or disable all product checked on table load.

  12. Set a class name for Table: Add custom class for custom design.

  13. Temporary Number for Table: Table ID when you need use it.

  14. (Add to cart) Text: Add custom add to cart button text.

  15. (Add to cart(Selected]) Text: Add custom add to cart selected button text.

  16. (All Check/Uncheck) Text: Add custom all check/uncheck label text.

  17. AuthorID/UserID/VendorID (Optional): If you want to show products from specific vendor.

  18. author_name/username/VendorUserName (Optional): If you want to show products from specific vendor.

  19. Product Type (Product/Variation Product): Choose when you want to show only variable product in a table with one click add to cart feature.

#Condition Tab

  1. Sorting/Order: Set the product table's shorting order.

  2. Order By: Select meta field to short product table.

  3. Set Minimum Price: Set the minimum price condition on product table.

  4. Set Maximum Price: Set the maximum price condition on product table.

  5. Description Type: Set the description type condition on product table.

  6. Only Stock Product: If you want to show only stock products.

  7. Only Sale Products: If you want to show only sale products.

  8. Post Limit/Per Load Limit: The maximum number of products you want to show in one page.

  9. Third Party Addons Supporting: Whenever you need to work other plugin which doesn't working with our product table.

#Search and Filter Tab

  1. Advance Search Box: Enable/disable advanced search form.

  2. Search From: Select the meta field for search query.

  3. Taxonomy Keywords for Advance Search Box (Separate with comma[,]): Add your custom taxonomies to add below.

  4. Choose [product_cat]: From previous step you can control this field to hide or show in search form.

  5. Mini Filter: Show/hide mini filter above of the table.

  6. Taxonomy Keywords for Filter (Separate with comma[,]): Add taxonomy to show up in mini filter.

#Design Tab

  1. Select Template: Design your table with our pre-defined template.(Work-in-progress)

  2. Table Customize: You can customize you product table using this design tab.

#Configuration Tab

  1. Basic Settings: From this section you can control your table specific configuration.

  2. Label Text: From this section you can change all the buttons, input fields, custom messages texts.

  3. External Plugin: From this section you can change external plugin YITH's label text.

  4. Table's Default Content: Change stock, backorder messages.

  5. All Messages: All other messages could be changed from here.

#Responsive Tab

  1. Mobile Column: From this column you can select which columns you want to show in the mobile devices.

  2. Tablet Column: From this column you can select which columns you want to show in the tablet devices.

  3. Drag and Drop Column: You can re-arrange all the items by simply drag and drop.

#FAQ

Can I search with SKU?

Ans:Yes, from version (6.0.7), You able to customized Advanced Search Field. It will only work from 6.0.7

Can I customize my table's design?

Ans:Yes, from version (4.4), You able to customized your table design. It will only work for latest version

Is my table's configuration is saved for use future?

Ans:Yes, from version (4.1) each table's configuration is saved as Table ID. You can edit any Table from the Table ID where your pre used table configuration remain same as before.

Will it work with old shortcode?

Ans:Yes, It's compatible with both old and new shortcode.

Is there any way, where I can duplicate or copy an existing PRODUCT_TABLE 's shortcode

Ans:Yes, You can use any Post Duplicator plugin. Don't worry about temporay_number of table's. Our plugin will automatically generate new one.

Can I show product's attribute in table column?

Ans:Yes, It's possible from V4.0 version. If you have old version, Atrributes column will not work.

Can I display product's variation in another column?

Ans:Yes, It's possible from V4.0 version. If you have old version, Variations column will not work. Note:

Is there a possibility to show product variations in the table?

Ans:Product variation automatically displays in action column where Add to cart button available.

How does Woo Product Table pro work?

Ans:The plugin lets you add product tables to any page, post or widget by Shortcode. To create a WooCommerce table view, simply choose the options for your product tables on the settings page(Dashboard -> Product Table->Add New Shortcode) and Publish the Shortcode.
You will get a table ID. Copy and paste that table id to your desired Page/Post.You can always contact us for expert advice on advance uses of Woo Product Table Pro.

Is there a possibility to show product variations in the table?

Ans:Yes, It's possible.

Can I choose which products are displayed?

Ans:Yes, there are lot of options to choose which products are included (or excluded) in the table. Select products by category, max-price, min-price, sort etc.

Can I customize the column headings?

Ans:Yes, you can customize heading of all the columns in the table. Also you can hide the table header if you want.

Will it work with my theme?

Ans:We have tested Woo Product Table pro with a range of free and premium themes and are confident that it will work perfectly with your theme.

Which version of WordPress is supported?

Ans:Woo Product Table is fully tested with the latest version of WordPress. We always recommend running the most up-to-date version of WordPress, but this plugin supports older installations

Is there any Video Tutorial?

Ans:Yes, there are few Youtube Video Tutorial available. Video link for Woo Product Table pro using and configuring.

I want to show different product table in different pages based on different categories. Any way?

Ans:Yes. Definitely there is a way. You can show your products in the table according to your selected categories. For doing this go to Dashboard->Product Table->Shortcode Generator) and click on the Basics tab.You will find there a field "Category Includes" . Just click on that field and select your category which you want show, and finally generate the shortcode.

How to set a custom class name in the table, so that, I can apply custom design on the product table?

Ans:Yes. You can set a class name for the table. For this Go to shortcode generator (Dashboard->Product Table->Shortcode Generator). Click on the Basics tab and find a field named "Set a Class name for Table" . Set your desired class name in the field. That's it.

How to install the update?

Ans:You can easily update our plugin. First, you've to deactive old version. Then, delete it. Now, reinstall again our latest version plugin and active it.

Is it possible to add some custom fields in the product table?

Ans:Yes, it's.You just add the custom field slug inside the shortcode with prefix of "cf_". For example, If your custom slug is color then the shortcode will be just like this, column_keyword='cf_color,thumbnails,product_title" column_title='Color,Thumbnails,Product Title" . Thus you can add custom field into your product table.

Is it possible to change Mini Cart's position of WPT Pro, or can I hide it?

Ans:Yes, position can be changed. Also you can hide, if you want. When you generate a shortcode, you will get a drop-down option named "Mini Cart Position" from shortcode generator's Basics Tab. There are three options named Top, Bottom or none. Select anyone from here.

Does this plugin support WooCommerce Product Add-Ons plugin?

Ans:Our plugin isn't compatible with WooCommerce Product Add-Ons plugin at this moment. We hope to add this in future.

How to show products in the table which are only in stock?

Ans:In our table "Only Stock" products shows by default. But, if you want to change that all products will be shown in the table then you need to change it in the shortcode. For doing this, go to shortcode generator (Dashboard->Product Table->shortcode Generator). After clicking on the conditions tab you will see a drop-down named "Only Stock Product", you can disable/enable from here. Or you can do it by editing the shortcode also. Just add the attribute and value only_stock="no" inside the shortcode. Thanks.

Ans:Yes, you can. To do it, check and enable "Short Message" column from Shortcode Generator and generate shortcode. (Dashboard->Product Table->Column) You can use your preferred column title then using "Short Message" in shortcode.

Click on the Basics tab and find a field named "Set a Class name for Table". Set your desired class name in the field. Use (mobile_hide='here column_keywords with coma') otherwise. That's it.

Is Thumbnails image re-sizable?

Ans:Yes, you're able to customize thumbs image for your product's table. Go to Configure page of our plugin (Dashboard->Product Table->Configure). Over there, you will find a input box called "Thumbs Image Size". Set your image size's dimension. Remember, here supports only int/numeric number. So don't insert px or inc etc. Only number. For example if you input 100 then image size will be 100px x 100px.

<?php

    /**
    * Taxonomy column generator
    * clue is: tax_
    *
    * @param type $item_key
    * @return String
    */
    function taxonomy_column_generator( $item_key ){
        $key = 'tax_';
        $len = strlen( $key );
        $check_key = substr( $item_key, 0, $len );
        if( $check_key == $key ){
        return $item_key;
        }
    }

    /**
    * Custom Fields column generator
    * clue is: cf_
    *
    * @param type $item_key
    * @return String
    */
    function customfileds_column_generator( $item_key ){
        $key = 'cf_';
        $len = strlen( $key );
        $check_key = substr( $item_key, 0, $len );
        if( $check_key == $key ){
        return $item_key;
        }
    }

    /**
    * Making new String/description based on word Limit.
    *
    * @param String $string
    * @param Integer $word_limit
    * @return String
    */
    function wpt_limit_words($string = '', $word_limit = 10){
        $words = explode(" ",$string);

        $output = implode(" ",array_splice($words,0,$word_limit));
        if( count($words) > $word_limit ){
        $output .= $output . '...';
        }
        return $output;
    }

        * Load Minicart Normally above at table, although we can set to bottom
        *
        * @param String ajax_url
        * @param boolean true_false
        * @returns void
        */
        function load_wpt_cart(ajax_url,true_false){
            $.ajax({
                type: 'POST',
                url: ajax_url,// + get_data,
                data: {
                    action: 'wpt_cart_auto_load',
                },
                success: function(data) {
                    if(data.search(config_json.mcart_empty_now) < 200){ //'Your cart is empty.'
                    $('.tables_cart_message_box').html(data);
                    }
                    if(true_false === 'yes'){
                    $('.tables_cart_message_box').html(data);
                    }
                },
                error: function() {
                },
            });
        }


        $('body').on('click','a.remove',function(){

        var setFewSeconds = setInterval(function(){
            load_wpt_cart(ajax_url, 'yes');
            clearInterval(setFewSeconds);
